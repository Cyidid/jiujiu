# 啾啾

啾啾是一个轻量 macOS 桌宠。它会漂浮在桌面上，支持拖拽、右键菜单、自由游走、跟随鼠标、角落休息、随机眨眼/跳跃/梳毛/睡觉、状态养成、心情气泡、专注计时，以及定时休息提醒。

## 设计方向

这版把主流桌宠常见做法落到一个很小的原生 macOS 应用里：

- 低打扰：默认是浮动小窗口，不抢 Dock，不遮挡正常工作流。
- 可互动：点击会有反馈，拖拽后会恢复自由游走。
- 有状态：啾啾会随机眨眼、跳跃、梳毛、睡觉，而不是静态贴图。
- 可养成：记录饱腹、开心、精力，支持喂小鱼干、陪它玩、摸摸头。
- 可表达：状态变化会用桌面气泡表达，不必频繁打断用户。
- 可控：右键菜单可以切换游走、跟随鼠标、角落休息，也能调大小和速度。
- 尺寸克制：默认按素材真实比例显示为小桌宠，避免挡住桌面内容。
- 动作质感：用独立图片层、地面阴影、悬浮呼吸、移动倾斜和点击弹性，让逐帧素材更顺、更有层次。
- 素材处理：运行素材已重绘为 `360x392` 透明画布的骨架动画帧，尾巴和爪子按动作参数运动，减少逐帧跳动和裁切。

## 素材说明

`additional/Documents/啾啾_设计素材/原始设计稿.png` 是 2048x2048 白底图，没有透明通道，不适合直接作为桌宠运行素材。当前应用使用 `additional/Applications/啾啾.app/Contents/Resources` 下重新绘制的逐帧 PNG。`scripts/generate_rigged_cat_sprites.py` 可重新生成这套骨架动画素材。
- 轻提醒：保留桌宠作为温和提醒媒介的思路，每 45 分钟提醒休息喝水，可开启勿扰模式。
- 专注陪伴：可以让啾啾进入角落休息，并启动 25 分钟专注计时。

## 构建

需要 macOS 和 Xcode Command Line Tools：

```bash
./scripts/build_app.sh
```

构建结果会更新仓库根目录的 `啾啾.app`。

## 文件结构

- `Sources/Jiujiu/main.swift`: 当前可维护源码。
- `啾啾.app`: 已构建的 macOS 应用包。
- `additional/`: 之前从本机归档的相关素材、旧 app、偏好文件和生成过程文件。
