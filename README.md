# 甜喵物语

甜喵物语是一个卡通动漫风格的轻量 macOS 桌宠。它会漂浮在桌面上，支持拖拽、右键菜单、自由游走、跟随鼠标、角落休息、随机眨眼/跳跃/梳毛/睡觉、状态养成、心情气泡、专注计时，以及定时休息提醒。

## 设计方向

这版把主流桌宠常见做法落到一个很小的原生 macOS 应用里：

- 低打扰：默认是浮动小窗口，不抢 Dock，不遮挡正常工作流。
- 可互动：点击会有反馈，拖拽后会恢复自由游走。
- 有状态：甜喵会随机眨眼、跳跃、梳毛、睡觉，并用独立关节完成动作。
- 可养成：记录饱腹、开心、精力，支持喂小鱼干、陪它玩、摸摸头。
- 可表达：状态变化会用桌面气泡表达，不必频繁打断用户。
- 可控：右键菜单可以切换游走、跟随鼠标、角落休息，也能调大小和速度。
- 尺寸克制：默认按素材真实比例显示为小桌宠，避免挡住桌面内容。
- 动作质感：头、躯干、前爪、后身和尾巴分层驱动；走路时双爪反相摆动，跳跃包含下蹲与收爪，梳毛由单爪抬到脸侧，睡眠使用独立呼吸和尾部动作。
- 互动反馈：点击时头部先看向触点，对应前爪和尾巴随后响应，身体通过缓入缓出的轻回弹恢复；拖拽结束使用独立落地动作，避免多套动画叠加。
- 素材处理：运行素材采用统一的 `360x392` 透明画布和重叠关节连接，不再通过整张动作图片轮回播放。

## 素材说明

当前应用使用 `Resources/` 下的六张透明骨骼部件 PNG 和 `AppIcon.icns`。多角度动漫源图保存在 `assets/source/multiview/`，`scripts/generate_rig_parts.py` 会根据 `normal.png` 生成动画部件，`scripts/generate_app_icon.py` 可重新生成应用图标。
- 轻提醒：保留桌宠作为温和提醒媒介的思路，每 45 分钟提醒休息喝水，可开启勿扰模式。
- 专注陪伴：可以让甜喵进入角落休息，并启动 25 分钟专注计时。

## 构建

需要 macOS 和 Xcode Command Line Tools：

```bash
./scripts/build_app.sh
```

构建结果会生成本地 `甜喵物语.app`。该目录是构建产物，不提交到仓库；发布版请从 GitHub Releases 下载 zip。

## 文件结构

- `Sources/TianMiao/main.swift`: 当前可维护源码。
- `Resources/`: 应用运行时使用的图片帧和 `AppIcon.icns`。
- `assets/source/multiview/`: 多角度动漫源图。
- `scripts/`: 构建、素材生成和图标生成脚本。
